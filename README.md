# JP IT STAFFING – Frontend (Job Portal UI)

JP IT STAFFING is a modern and responsive job portal frontend designed to connect skilled IT professionals with companies looking for quality talent. This project focuses on creating a clean, scalable, and user-friendly interface that simulates a real-world staffing and recruitment platform.

---

## 🚀 Live Demo

🌐 Live Site:- https://jp-it-staffing.netlify.app/

---

## 📌 Project Overview

This project represents the frontend of a staffing and recruitment platform. It allows users to explore job opportunities, view company services, and understand the hiring process through a structured and intuitive interface.

The focus of this project is on:
- Clean UI/UX design
- Functional job search and filtering
- Realistic structure of a professional staffing platform
- Scalable frontend architecture

---

## 🎯 Key Features

- Modern and responsive user interface
- Job listing page with:
  - Search by job title
  - Filter by location
  - Filter by job type
- Dedicated pages:
  - Home
  - Jobs
  - Services
  - About
  - Contact
- Clean and reusable component-based architecture
- Responsive design for desktop, tablet, and mobile
- Smooth navigation using client-side routing

---

## ▶️ Getting Started (Local Setup)

### 1. Clone the repository
git clone :- https://github.com/Priyanshu87571/Job-Portal-UI.git

### 2. Navigate to the project directory
cd Job-Portal-UI

### 3. Install dependencies
npm install

### 4. Start the development server
npm run dev

The application will run locally at:
http://localhost:5173

---

## 📦 Build for Production

npm run build

The optimized production build will be generated in the `dist` folder.

---

## 🌐 Deployment

This project is deployed using Netlify.

Build Settings:
- Build Command: npm run build
- Publish Directory: dist
